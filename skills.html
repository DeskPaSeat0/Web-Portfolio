Time-Out: Network Control Application

Time-Out is a mobile application designed to help parents limit and monitor the internet usage of their children on their Android devices. It uses an unconventional approach that does not rely on any existing device management or network restricting software on the market. This application was built using Java, Kotlin, and Firebase for the database.
